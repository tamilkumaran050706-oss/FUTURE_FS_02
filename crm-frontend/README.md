# CRM Frontend (React + Vite + Tailwind CSS)

Real Estate Lead Management CRM client application.

## Setup Instructions

1. Navigate to the frontend directory:
   ```bash
   cd crm-frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build the application:
   ```bash
   npm run build
   ```

## Available Scripts

In the project directory, you can run:

- `npm run dev`: Runs the app in development mode at `http://localhost:5173`.
- `npm run build`: Builds the production-ready assets into the `dist` folder.
- `npm run lint`: Lints the source files for errors and standard code quality.
- `npm run preview`: Locally previews the production build.
