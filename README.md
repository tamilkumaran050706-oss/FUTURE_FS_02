# FUTURE_FS_02: Real Estate Lead Management CRM

Task 2 - Real Estate Lead Management CRM built using React, Node.js, Express, and MySQL.

## Project Overview
This project is a web-based Real Estate Lead Management CRM designed to capture, organize, and track real estate client leads. It provides a clean dashboard for administrators and agents to manage leads throughout the sales funnel.

## Tech Stack
- **Frontend:** React + Vite, Tailwind CSS, Axios, React Router DOM, Lucide React
- **Backend:** Node.js, Express
- **Database:** MySQL

## Folder Structure
```
FUTURE_FS_02/
├── crm-frontend/        # React SPA Frontend
└── crm-backend/         # Express API Backend
```

## Development Roadmap
1. **Phase 1: Project Setup and Boilerplate (Current)**
   - Initialize frontend React/Vite/Tailwind app.
   - Initialize backend Express application.
   - Configure project structure and configurations.
2. **Phase 2: Database Schema & API Development**
   - Create MySQL schema for leads.
   - Implement backend CRUD endpoints (GET, POST, PUT, DELETE leads).
   - Add backend request validations and global error handlers.
3. **Phase 3: Frontend Layout & Routing**
   - Build MainLayout, Navigation, and Sidebar layout.
   - Set up page routing (Dashboard, Leads, Add/Edit Lead).
4. **Phase 4: API Integration & UI Components**
   - Integrate Axios client with backend endpoints.
   - Create tables, status tags, form inputs, and dashboards.
5. **Phase 5: Refinement & Polish**
   - Add filtering, search, and sorting.
   - Refine design aesthetics and animations.
